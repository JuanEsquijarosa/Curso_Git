# Curso de Git

## Ejercicio #1

1. Crear una cuenta de GitHub.

2. Crear un repositorio.

3. Modificar el repositorio y añadir información sobre sus intereses.

4. Enviar enlace del repositorio a acg@albertocg.com

## Ejercicio #2

1. Clonar el repositorio acgissues/Curso_Git

2. Crear una rama.

3. Crear un archivo (*.txt) con tu nombre en la carpeta Peliculas.

4. Subir la rama al repositorio y crear un Pull Request para añadirla.
